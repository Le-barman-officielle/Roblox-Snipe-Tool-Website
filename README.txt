# 🔴 Roblox Snipe Tool // by Kinito.exe

> **INTELLIGENCE & RECON SYSTEM — SCAN. TARGET. ACQUIRE.**

Roblox Snipe Tool est un outil avancé permettant d’explorer et d’analyser en profondeur n’importe quel profil Roblox via une interface moderne et intuitive.

---

## 🎯 Target System

Plusieurs méthodes pour trouver une cible :

- `// TARGET BY USER ID`  
  ➤ Entrez un ID et cliquez sur **[ ACQUIRE TARGET ]**

- `[ RANDOM TARGET ]`  
  ➤ Sélectionne un utilisateur aléatoire

- `// SCAN BY USERNAME`  
  ➤ Recherchez un joueur avec **[ SCAN USERNAME ]**

---
